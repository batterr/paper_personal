# 贴纸人格题库与计分配置 V4.1

> 适用范围：独立网站 / 手机 H5
> 版本方向：基于真实评论与贴纸导出数据，混合正式题、搞怪题、随机题
> 核心目标：既像真实贴纸玩家，又保留整活感和传播感
> 当前线上代码源：`web/src/data/questions.ts`

## 1. 数据来源与本次优化

- 已参考真实评论数据：`database_export-paper-3gvhj92f6e8cf0c9-review.json`
- 已参考真实贴纸数据：`database_export-paper-3gvhj92f6e8cf0c9-sticker.json`
- 真实高频信号已映射进题库：`平均分 / 评论数 / 想要人数 / 拥有人数 / 已贴人数`
- 真实标签已映射进题库：`参考图 / 排废体验 / 模切工艺 / 气味情况 / 素材拆分 / 素材排版 / 定位完整度 / 定位方式 / 用时 / 白墨 / 色差`
- 评论文风也有吸收：如“留胶”“遮挡看不清”“很爽贴”“有小鼻嘎”“图很美但体验一般”等真实表达

## 2. 本版题库结构

- 主线题总数：`40`
- 抽题方式：每次从 `40` 题里随机抽 `30` 题
- 题型分层：
  - `正式题`：正式合理，和真实数据强关联
  - `搞怪题`：搞怪、夸张、适合整活
  - `随机题`：随机脑回路题，允许无厘头
- 当前分布：`正式题 20 / 搞怪题 11 / 随机题 9`

## 3. 六个显性维度

- `T`：囤货执念
- `Q`：安利控场
- `R`：贴用代入

## 4. 评分规则

- 每题可以是 `2 / 3 / 4` 个选项，不再强制四选一
- 选项可以是普通反应题，也可以是 `赞同 / 不赞同` 这种布尔或近似布尔题
- 部分题允许用户选择“不适用，跳过”，跳过时不加分
- 前端展示时会随机打乱选项顺序，并隐藏内部 `A / B / C / D` 计分键，避免用户感到“最后一个就是最优解”
- 每个选项仍可对所属维度加 `0 / 1 / 2 / 3` 分，但不要求每题都覆盖完整梯度
- 抽题时会保持六维覆盖平衡，每次每个维度抽 `5` 题
- 隐藏加问题继续只用于放大极端人格，不改变主线结构
